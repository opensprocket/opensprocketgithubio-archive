How it Works

This page is styled to have up 1000px in width, after which the margins will automatically increase to keep the body centered. This and additional padding for the section helps keep the image under the maximum size available, which is 800px.

Below, we have an image that is responsive to the width of the viewport. By using the img tag with the srcset and sizes attributes, the image below will only download and display the size appropriate to the viewport.

There are breakpoints at the following widths:

  • 320px
  • 420px
  • 600px
  • 800px and up

Open up your browser inspection tool (Ctrl+Shift+I), turn on your mobile emulator and pick a device.

Then switch to the network tab and reload the page. You should see the image closest to that resolution and pixel density being downloaded. If you check the "Disable Caching" box, you can switch sizes without having to clear the cache after reloading.

For more information regarding this, please visit the MDN Documentation for responsive images.
