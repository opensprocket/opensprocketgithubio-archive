Typography

This section of the planning document will cover the types of font used and the respective sizes for heading and paragraph elements.

The default heading font will be Patua One, a serif font. This font was chosen because of large block lettering and its readability.

IBM Plex Sans was chosen as the default text and paragraph font. This is mainly because sans-serif fonts are easier to read on a screen than a serif font. In addition to that, the font was chosen because of the larger kerning space which is present in the font by default.
