Color Palatte

PrimarySecondary
[#333333][#23e2ef]

The primary colors will be applied to things such as headers and large sections of the page that will be in color.

Since these colors are darker, they will be used as a background color. The text in these areas will be white to maintain readability.

Accent 1Accent 2
[#b0dd8d][#dcbf85]

Accent colors are lighter in nature, and will be used sparingly throughout the page. Accent 1 will be used for the hover effect on the navigation.

Accent 2 will be used as the default color for whitespace. Because it is slightly off-white, it will not be used for text on a lighter background.

Instead, it will be used in combination with CSS element states (such as hover and active).
